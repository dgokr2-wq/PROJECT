
# Учебное приложение шифрования SAFER K-128 (Python + Tkinter)

Проект реализует учебную (облегчённую) версию алгоритма блочного шифрования
по мотивам SAFER K-128 и графический интерфейс на `tkinter` для шифрования
и расшифрования текстовых данных.



## Запуск

1. Установить зависимости:

   ```bash
   pip install -r requirements.txt
   ```

2. Запустить GUI:

   ```bash
   python main.py
   ```

## Использование

1. Выбрать один из датасетов в верхней части окна и нажать **«Загрузить датасет»**.
2. При необходимости нажать **«Копировать датасет в текст»**, чтобы вставить содержимое
   таблицы в поле «Входной текст».
3. Ввести ключ (строка, приводится к 128 бит) в поле «Ключ».
4. Для шифрования:
   - в поле «Входной текст» должен быть обычный текст;
   - нажать кнопку **«Зашифровать»**;
   - результат (Base64) появится справа в поле «Результат».
5. Для расшифрования:
   - в поле «Входной текст» должен быть шифртекст (Base64);
   - нажать кнопку **«Расшифровать»**;
   - исходный текст появится справа.

## Важно

Реализация носит учебный характер и **не предназначена для промышленного использования**
в качестве криптографического средства.
